# 📱 Limitless Mobile Application

Limitless Mobile is a **cross-platform mobile application built with Flutter** that connects to the **Limitless Symfony backend platform**.

The application provides users with a mobile interface to interact with the platform's services, including account management, QR code features, and real-time data access.

---

# 🚀 Features

- 📱 Cross-platform mobile app (Android / iOS)
- 🔐 Secure authentication system
- 🔄 API communication with Symfony backend
- 📷 QR code scanning and generation
- 📊 Data visualization and user interaction
- ⚡ Responsive and modern UI
- 🔗 Real-time data synchronization with server
